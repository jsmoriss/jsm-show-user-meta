Description

Wondering about the user meta your theme and/or plugins might be creating?

Want to find the name of a specific user meta key?

Need some help debugging your user meta?

The JSM's Show User Meta plugin displays all user meta (aka custom fields) keys and their unserialized values in a metabox at the bottom of user profile editing pages.

There are no plugin settings — simply install and activate the plugin.

Installation

Automated Install

  1. Go to the wp-admin/ section of your website.
  2. Select the Plugins menu item.
  3. Select the Add New sub-menu item.
  4. In the Search box, enter the plugin name.
  5. Click the Search Plugins button.
  6. Click the Install Now link for the plugin.
  7. Click the Activate Plugin link.

Semi-Automated Install

  1. Download the plugin archive file.
  2. Go to the wp-admin/ section of your website.
  3. Select the Plugins menu item.
  4. Select the Add New sub-menu item.
  5. Click on Upload link (just under the Install Plugins page title).
  6. Click the Browse... button.
  7. Navigate your local folders / directories and choose the zip file you downloaded previously.
  8. Click on the Install Now button.
  9. Click the Activate Plugin link.

Developer Filters

'jsm_sum_view_cap' ( 'manage_options' ) — The current user must have these capabilities to view the "User Meta" metabox (default: 'manage_options' ).

'jsm_sum_screen_base' ( true, $screen_base ) — Add the "User Meta" metabox to the editing page for user screen base (example: 'user-edit', 'profile').

'jsm_sum_user_meta' ( $user_meta, $user_obj ) — The user meta array (unserialized) retrieved for display in the metabox.

'jsm_sum_skip_keys' ( $array ) — An array of key name regular expressions to ignore (default: '/^closedpostboxes_/', '/columnhidden$/', '/^meta-box-order_/', '/^metaboxhidden_/', and '/^screen_layout_/' ).
